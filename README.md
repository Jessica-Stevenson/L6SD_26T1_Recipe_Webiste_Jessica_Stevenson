Dough-lightful Recipes website

This is a website that allows users to create and share recipes of their own creation while checking out other recipes that other users have made.

Uses:
Django and Pillow

Upon opening visual studio code here are the commands to run:

- python -m venv recipe
- recipe\Scripts\activate.bat
- pip install Django
- pip install Pillow
- cd recipesite

And to run the website:

- python manage.py runserver
Then head to http://127.0.0.1:8000/
